# HoMM3-MapGen-v2

## Init submodules
`git submodule update --recursive --init`

## Enable pre-commit:
```bash
chmod +x ./scripts/pre-commit.sh
ln -s ./scripts/pre-commit.sh .git/hooks/pre-commit
```
